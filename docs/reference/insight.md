# `nm insight` commands

Index: [index.md](index.md)

| Command | Description |
|---|---|
| `nm insight weekly` | Weekly listening insight |
| `nm insight monthly` | Monthly-style insight from available history |
| `nm insight yearly` | Yearly-style insight from available history |

```bash
nm insight weekly --output json
nm insight yearly --output markdown
```

Insights use NetEase listening records and require login when `--uid` is omitted.
