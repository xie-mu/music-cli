# `nm album` commands

Index: [index.md](index.md)

## Commands in this group

| Command | Description |
|---|---|
| `nm album show` | View album details and track list |
| `nm album list` | List subscribed albums |
| `nm album sub` | Subscribe to an album |
| `nm album unsub` | Unsubscribe from an album |
| `nm album dynamic` | Album dynamic data (comments, shares) |
| `nm album summary` | Album summary |
